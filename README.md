# tom-riddle
An interactive HTML5 Tom Riddle Diary for iPad and web.
How to Set This Up on Your iPad
Option A: Quick Local Test (Using an App)
Download a free code editor app on your iPad like Koder or Textastic, or save the file to your iCloud Drive and use an HTML viewer app.

Create a new file called riddle.html, paste the code above, and tap Preview/Run.

Option B: The Full-Screen "App" Experience (Recommended)
To make it feel like a real native iPad app without Safari URL bars or buttons:

Host the HTML file online for free using a platform like GitHub Pages, Netlify, or Vercel (just upload the single riddle.html file and rename it to index.html).

Open your live website link in Safari on your iPad.

Tap the Share button (the square with an upward arrow) in the Safari toolbar.

Scroll down and select Add to Home Screen.

Name it The Diary. Now, when you launch it from your iPad home screen, it will open in full-screen mode like a standalone app!
